JavaScript数据类型

JavaScript 数据类型

在 JavaScript 中有 6 种不同的数据类型:

  • string

  • number

  • boolean

  • object

  • function

  • symbol

三种对象类型:

  • Object

  • Date

  • Array

两个不包含任何值的数据类型:
  • null
  • undefined

typeof 操作符

可以使用 typeof 操作符来查看 JavaScript 变量的数据类型。

constructor 属性

constructor 属性返回所有 JavaScript 变量的构造函数。
