javascript中null和undefined的区别

null

在 JavaScript 中 null 表示什么都没有。null 是只有一个值的特殊类型，所以用来表示一个空对象引用。

undefined

在 JavaScript 中, undefined 是一个没有设置值的变量。

JavaScript 数据类型

在 JavaScript 中有 6 种不同的数据类型:

  • string

  • number

  • boolean

  • object

  • function

  • symbol

三种对象类型:

  • Object

  • Date

  • Array

两个不包含任何值的数据类型:
  • null
  • undefined

可以使用 typeof 操作符来查看 JavaScript 变量的数据类型。
